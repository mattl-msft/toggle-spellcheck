# Icon Placeholder Files

This extension requires icon files in PNG format. You can:

1. **Use an online tool** like [Figma](https://figma.com), [Canva](https://canva.com), or [Photopea](https://photopea.com) to create simple icons
2. **Use an AI tool** to generate icons
3. **Download free icons** from [Icons8](https://icons8.com) or [Flaticon](https://flaticon.com)

## Required Icon Files:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Simple Icon Idea:

A simple "Aa" text on a colored background works well. Use:

- Blue (#4285f4) for a professional look
- Green (#34a853) for an active/on state theme

## Temporary Solution:

The extension will work without icons, but the browser will show a default placeholder icon.

## Quick SVG to PNG Conversion:

If you have the SVG file (`icon.svg`), you can convert it using:

- Online tools like [CloudConvert](https://cloudconvert.com)
- Image editors like GIMP, Photoshop, or Paint.NET
- Command line tools like ImageMagick
